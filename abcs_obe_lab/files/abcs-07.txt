The illustration shows the action of searching for and adding a table component to the Book Catalog form of the Books Tracker application on the Application Designer page.

The Application Designer page contains 2 sections: 
* UI components section
* Edit canvas section

The UI components section contains the tabs toolbar and the components panel. 
The tabs toolbar contains 3 icons: 
* UI Components button (This button is selected.)
* Data Components button
* UI Themes button

The components panel contains 1 field and 1 area:
* Filter field: table (This field is selected.)
* Components area: Contains 1 table icon representing the Table UI element. (This icon is selected.).

The edit canvas section contains the Book Catalog form, which contains a table. The table contains 2 sample columns and 3 rows. The column names are “Column,” and the row values are “Value.”
