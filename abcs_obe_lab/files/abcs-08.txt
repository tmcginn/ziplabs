The illustration shows the Data step in the Table Creation Section of the Application Designer page. This step is the first step of the Table Creation wizard.

This section is used to define the data source and bindings of the table created for the Book Catalog application page on the Application Designer page.

The Table Creation section contains 4 buttons, 1 progress bar, 1 field, and 1 tile:
* Cancel button
* Next button
* Back button
* New Business Object button
* Table Creation progress bar: This bar contains the following steps: Data (selected), Mapping, Object Actions, and Options.
* Filter field: Filter Business Objects
* Book_Catalog_BO tile: This tile represents the previously created Business Object labeled Book_Catalog_BO (selected).
