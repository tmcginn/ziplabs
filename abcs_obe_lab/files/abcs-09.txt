The illustration shows the Mapping step on the Table Creation Section of the Application Designer page. Mapping is the second step in the Table Creation wizard.

The Table Creation section contains 5 buttons, 1 progress bar, 1 tile, and 3 fields:
* Cancel button
* Finish button
* Back button
* Next button
* Table Creation Progress bar: This bar contains the following steps: Data, Mapping  (selected), Object Actions, and Options.
* Business Object tile: Book_Catalog_BO Fields (selected)
* Filter Fields field
* New Field button
* Available fields: Id, Author (selected), Book Title, Created By, Creation Date, ISBN (selected), Last Update Date, Last Updated By, Published Date
* Selected fields: Id, Book Title, Author, ISBN, Published Date
