The illustration shows the Image Gallery tab on the Application Designer page.

There are 3 sections: The Categories section, the Image List section, and the Button section.

The Categories section contains a panel with 5 links:
* Functional
* Status
* Application Menu (selected)
* Custom
* All

The Image List section 2 fields:
* Filter field
* Image field: Lists all images corresponding to the selected category (Application Menu).

The Button section contains:
* Select button to select the image and close the Image Gallery page.
* 4 images representing the different states of the selected image: Enabled, Hover, Active, and Disabled.
