This illustration shows the Create Application window. It contains a form to create an application on Oracle Application Container Cloud Service. The form is divided in two sections vertically stacked: Application and Instances. 

The Application section contains the following fields: 

Name. It's a text field and it contains the BookRestService text.

Metering Frequency. It's a drop-down field and Monthly is selected.

Application Artifacts. Upload it's a check button and it's selected.

Archive. It contains a Browse button to select a file.

Manifest: It contains a Browse button to select a file.

Development Plan: It contains a Browse button to select a file.

Notes. It's a text field to enter a description for the application.
