This illustration shows the Create Application dialog box. It contains a form to create an application in Oracle Application Container Cloud Service. 

The Application section contains the following fields: 

The Name field contains the EmployeeService name.

Metering Frequency is a drop-down list and Monthly is selected.

Application Artifacts has an Upload option, which is selected.

Archive, Manifest, and Development Plan each contain a Browse button to select a file. The Browse button next to Archive is selected.
